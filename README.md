# 心灵重建 · EMDR + CBT 融合疗愈游戏

> 一款融合 **EMDR**（眼动脱敏与再处理）与 **CBT**（认知行为疗法）的网页互动小游戏，帮助用户识别认知偏差、安全处理创伤记忆、重建积极的自我认知。

## 🧠 设计框架

### 治疗理论基础

| 模块 | 对应疗法 | EMDR 阶段 |
|------|---------|-----------|
| 安全评估 | EMDR | Phase 1-2（历史采集 + 准备）|
| 目标识别 | EMDR | Phase 3（评估）|
| 认知偏差识别 | CBT | 认知扭曲心理教育 |
| 认知重构 | CBT | 思维记录单 |
| 双侧刺激脱敏 | EMDR | Phase 4（脱敏）|
| 正向安装 | EMDR | Phase 5（安装）|
| 身体扫描 | EMDR | Phase 6（身体扫描）|
| 接地气 / 安全地带 | EMDR + CBT | Phase 7（结束）|
| 会话总结 | EMDR | Phase 8（再评估）|

### 核心工具

- **SUD（主观痛苦单位）**：0–10 量表，贯穿全流程跟踪情绪强度变化
- **VOC（认知有效性）**：1–7 量表，衡量正向认知的可信度
- **双侧视觉刺激**：左右移动的光球，配合可选双侧音调（Web Audio API）
- **CBT 思维记录单**：热点思维 → 支持证据 → 反对证据 → 平衡性思维
- **5-4-3-2-1 接地气**：随时可调用的紧急情绪调节工具
- **成就徽章系统**：激励用户完成各阶段练习

## 🚀 如何运行

无需安装任何依赖，直接在浏览器打开：

```bash
# 方法一：直接打开文件
open index.html

# 方法二：用本地服务器（推荐，避免某些浏览器的安全限制）
npx serve .
# 或
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 📋 认知偏差类型（CBT 教育内容）

游戏涵盖以下 8 种常见认知扭曲：

1. **非黑即白**（全有或全无思维）
2. **灾难化**（小事放大成最坏结局）
3. **揣摩他人**（假设知道他人的负面想法）
4. **过度泛化**（从单一事件推断普遍规律）
5. **情感推理**（以感受代替事实）
6. **"应该"句式**（严苛的规则性要求）
7. **个人化**（把外部事件归咎于自己）
8. **心理过滤**（只关注负面细节）
